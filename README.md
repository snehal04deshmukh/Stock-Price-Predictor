# 📈 Stock Price Predictor

A Machine Learning and Deep Learning project that predicts stock closing prices using historical market data. The project leverages an LSTM (Long Short-Term Memory) neural network to capture time-series patterns and forecast future stock prices.

## 🚀 Features

- Historical stock price analysis
- Data preprocessing and normalization
- LSTM-based deep learning model
- Training and validation visualization
- Actual vs Predicted stock price comparison
- Performance evaluation using regression metrics

## 🛠 Tech Stack

- Python
- TensorFlow / Keras
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- yFinance

## 📂 Project Workflow

1. Collect historical stock data
2. Preprocess and normalize data
3. Create training sequences
4. Train LSTM model
5. Predict future stock prices
6. Evaluate model performance
7. Visualize actual vs predicted prices

## 📊 Evaluation Metrics

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
